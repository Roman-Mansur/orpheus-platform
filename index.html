Platform Overview

Audio collection for voice actors: scripted tasks, record, submit, review, deliver.

Roles

  • Expert - claim, verify transcript, record, submit
  • Reviewer - initial peer review + checklist
  • Admin/HDM - final review, batches, delivery, audit

Decisions

  • Human transcript verification (same expert, combined with recording)
  • Each task JSON provides both original transcripts: non-normalized + normalized
  • Locale-restricted tasks
  • Auto-QC on every submit: noise reduction ? peak normalization ? VAD trim (ms removed shown) ? ASR/WER ? SNR ? UTMOSv2 (TBD) ? Gemini artifact detection (TBD)
  • Flags are advisory; re-record recommended can be overridden by expert
  • Flagged tasks surface in reviewer QC Flags tab (high priority)
  • Raw + processed audio stored on submit
  • Final review: HDM/SPL when flagged/edge case
  • Screener: client set, random by locale

Orpheus Trainer Guide (key points)

  • ~10 sec recordings; read transcript exactly as written (zero tolerance)
  • Revise if unnatural; verify grammar, gender conjugations, localization
  • Upward inflection on questions; upbeat/neutral customer service tone
  • Test recording before batch; WAV 24kHz min (platform target: mono 16-bit 48kHz)
  • Screener gate: client approves before production tasking; resubmit v2/v3 on rework

Screener flow

  • Random script from client pool matched to expert locale
  • Same UI as production: both transcript variants, equipment check, record, submit
  • Tracks attempts + client feedback (replaces Google Form / Excel)
  • Task Dashboard locked until Client Approved = Yes

Auto-QC pipeline (on every submit)

  1. Noise reduction, peak normalization (on-platform)
  2. VAD trim � clip to speech boundaries; show ms removed at head and tail (sanity check that clip wasn't over-trimmed)
  3. ASR verification - WER vs revised or original transcript
  4. SNR verification - flag if <= 15.0
  5. UTMOSv2 naturalness scoring TBD @Sina
  6. Gemini artifact detection TBD @Sina

Flags are advisory. Re-record recommended can be overridden by expert. All flagged tasks go to reviewer QC Flags tab (high priority). Raw + processed audio stored.

Metadata fields

speaker_id, language, locale, transcript_id, script_batch, category, duration_s, transcript (non-normalized), normalized_transcript, transcript_revision, revised_transcript, revised_normalized_transcript, age_range, gender, native_language.

{
  "id": "TX-10001",
  "language": "English",

Both original transcript variants are provided by client at ingest. ASR QC compares against revised (or original non-normalized if unmodified).
